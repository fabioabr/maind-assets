Templates de e-mail Maind Tech

Templates HTML otimizados para Gmail (table-based, estilos inline, fallbacks de cor sólida). Pré-visualize, abra standalone, ou copie o HTML pra usar.

Como usar no Gmail
  1. Assinatura: abra pub/sign.html no navegador → selecione tudo (Ctrl+A) → copie (Ctrl+C) → cole em Gmail → Configurações → Geral → Assinatura. As imagens (pg-left/mid/right.png + cert/) precisam estar hospedadas em URL pública.
  2. Wrapper / Newsletter: abra o arquivo no navegador → selecione tudo → copie → cole no corpo de um novo e-mail (Compose). O Gmail preserva o formato visual.
  3. Para envio em massa (newsletter), use ferramenta tipo YAMM, Mailchimp ou SendGrid e faça upload do HTML bruto. Visualize via view-source: no navegador (Ctrl+U) pra copiar o código.
  4. Outlook desktop não renderiza gradientes CSS, background-clip:text, nem múltiplos backgrounds — vai cair em bgcolor sólido / color sólido (já incluso como fallback).
  5. Dark mode automático do Gmail mobile pode inverter cores dos templates light. Se for problema, prefira o template dark.
